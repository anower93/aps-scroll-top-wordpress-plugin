== Description ==

AP Scroll To is a simple plugin. It is easy to use ever. You have to install and activate just! No needs any settings! It is free and it will always be free! Let's start Scrolling!

= Docs & Support =

I believe that it won't need any support. Just install, it will work easily. If you were unable to find the answer to your question on the FAQ or in any of the documentation, you should check the [support forum](https://wordpress.org/support/plugin/ap-scroll-top/) on WordPress.org. If you can't locate any topics that pertain to your particular issue, post a new topic for it.


= Privacy Notices =

You can you use this only personal purpose.


= Recommended Plugins =

It's don't need any recommended plugin.

== Installation ==

1. Upload the entire `ap-scroll-top` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

Enjoy your smooth scrolling. Have fun!
